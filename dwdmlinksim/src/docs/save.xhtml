.SAVE Lines

General form:

.SAVE vector vector vector ...


Examples:

.SAVE i(vin) input output
.SAVE @@m1[id]

The vectors listed on the .SAVE line are recorded in the rawfile for use later with ngspice or nutmeg (nutmeg is just the data-analysis half of ngspice, without the ability to simulate). The standard vector names are accepted. If no .SAVE line is given, then the default set of vectors are saved (node voltages and voltage source branch currents). If .SAVE lines are given, only those vectors specified are saved.
